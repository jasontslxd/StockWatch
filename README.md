# React app with Typescript

## Setting up the dev environment
  ```sh
  npm install -g bun
  bun install 
  ```

## To start dev server
  ```sh
  bun dev
  ```

## To commit
  ```sh
  bun commit
  ```

## To deploy
  ```sh
  bun install -g firebase-tools
  firebase login
  bun run build
  firebase deploy
  ```
